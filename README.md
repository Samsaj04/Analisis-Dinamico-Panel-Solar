# Herramienta para el Análisis Dinámico de un Panel Solar

## Descripción

Este proyecto desarrolla un modelo computacional en Python para estudiar el comportamiento dinámico de un panel solar en voladizo sometido a vibraciones durante una misión espacial.

El programa permite:

* Calcular frecuencias naturales de vibración.
* Obtener formas modales.
* Evaluar el factor de magnificación dinámica.
* Analizar la distribución relativa de esfuerzos flectores.
* Realizar estudios paramétricos variando materiales y espesores estructurales.

## Software Utilizado

* Python 3
* NumPy
* SciPy
* Matplotlib

## Ejecución

Ejecutar el archivo principal:

```bash
python panel_especial.py
```

Las figuras empleadas en el informe se generan y almacenan automáticamente.
